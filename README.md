


# **LangGraph-Based Chatbot with Memory**



## Overview
A conversational AI chatbot built using LangGraph that maintains memory and context across multiple user interactions.

## Problem Statement
Most chatbots fail to remember previous interactions, leading to poor conversational experience. This project introduces memory-aware responses using stateful workflows.

##  Architecture
User Input → LangGraph State → Memory Store → LLM → Context-Aware Response

## Tech Stack
- Python
- LangGraph
- LangChain
- LLM APIs (Groq/OpenAI)

##  Features
- Stateful memory handling
- Multi-turn conversations
- Agent-based workflow design
- Context-aware response generation

  
- Developed a chatbot with memory for multi-turn, context-aware conversations
